# CRM Local

Sistema de CRM (Customer Relationship Management) desktop desenvolvido em Java para gerenciamento local de clientes, contatos e negociações.

## 📋 Requisitos

- Java 11 ou superior
- Maven 3.8 ou superior (para compilação)

## 🚀 Como executar

### Usando Maven:
```bash
# Compilar
mvn clean compile

# Executar
mvn exec:java -Dexec.mainClass="com.crm.Main"
